# Planificación de desarrollo – Casos de uso MVP Styla

Este documento detalla la planificación de desarrollo para los próximos sprints, indicando los casos de uso seleccionados para cada uno. Los identificadores UCxxx corresponden al listado de casos de uso del documento principal. No se incluyen los módulos de autenticación (ya desarrollado en el anterior sprint) ni planificación de outfits (descartado para mvp).

---

## Casos de uso seleccionados

### Gestión del Guardarropa

UC005. Agregar prenda al guardarropa
El usuario captura o selecciona una foto de una prenda desde su galería y la clasifica en una categoría específica (camisa, pantalón, zapatos, accesorios). Este caso de uso procesa la imagen, la almacena y crea un registro de la prenda en la base de datos vinculada al usuario.

UC006. Etiquetar prenda
El usuario asigna etiquetas a una prenda existente en su guardarropa, especificando atributos como color, estilo y ocasión de uso. Esto permite filtrar y recomendar prendas de manera más precisa.

UC007. Visualizar guardarropa
El usuario accede a la sección de guardarropa para ver todas sus prendas organizadas en cuadrícula o lista.

UC008. Filtrar prendas
El usuario aplica filtros (categoría, color, estilo, ocasión) para encontrar rápidamente lo que busca en su guardarropa.

UC009. Editar información de prenda
El usuario puede modificar la categoría, etiquetas o foto de una prenda existente.

UC010. Eliminar prenda del guardarropa
El usuario elimina una prenda que ya no posee o no desea incluir en recomendaciones.

### Comunidad y Social

UC011. Crear post con outfit
El usuario publica un outfit en la comunidad con foto y descripción.

UC012. Visualizar feed de comunidad
El usuario explora los outfits publicados por otros usuarios en un feed.

UC013. Dar like a post
El usuario da like a un post del feed comunitario.

UC014. Comentar post
El usuario comenta en un post de la comunidad.

UC015. Guardar outfit de inspiración
El usuario guarda un outfit publicado por otro usuario en su colección personal de inspiración.

UC016. Visualizar perfil de otro usuario
El usuario navega al perfil público de otro miembro de la comunidad.

UC017. Seguir/Dejar de seguir usuario
El usuario sigue o deja de seguir a otro miembro de la comunidad.

UC018. Ver outfits guardados como inspiración
El usuario accede a su colección personal de outfits guardados.

### Recomendaciones con IA

UC019. Obtener recomendación rápida de outfit
El usuario recibe una sugerencia inmediata de outfit basada en su guardarropa y preferencias.

UC020. Generar recomendación personalizada con prompt
El usuario describe el tipo de outfit que desea y recibe una recomendación personalizada.

UC021. Solicitar recomendación por tags
El usuario selecciona tags específicos para recibir recomendaciones filtradas.
